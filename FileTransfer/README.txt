README for FileTransfer python project.
Author: Lateef Adetona
Date: December 2, 2018

How to use project:
1. Run server.py
2. Run client.py
3. On client.py, enter the hostname that is given on the sevrer
4. On client.py, enter the name for the file you are about to receive, along with the file extension
5. On server, enter the name of the file you want to transfer.
